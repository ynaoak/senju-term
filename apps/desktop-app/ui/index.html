スレッド

    ワークフロー

    よく使うコマンドをテンプレート化。検索・ショートカット・クイックボタンから実行できます。

      SSH 接続

      接続先を登録。新規スレッド作成時のメニューや下のリストから接続できます。パスワードは保存されません。

        ターミナルプロファイル

        起動するシェルをプロファイルとして管理します。★ が「＋ 新規」で起動する既定です。

          起動セット

          いつも開くシェル・SSH 接続・ワークフローをまとめて登録し、ワンクリックで一括起動できます。

            設定

            外観・ターミナル・起動の設定。変更は「保存」で反映され、開いているスレッドにも即時適用されます。

            外観
            ターミナル

            bash / zsh / fish の起動時に OSC 133 のプロンプト連携を自動で組み込みます。rc ファイルは変更されません。

            起動

            設定・登録データは JSON ファイルとしてアプリ設定ディレクトリに保存されます。
