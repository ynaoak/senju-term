# docs

`apps/desktop-app` と `apps/web-lp` の**両方に関わる共通ドキュメント**を置く場所です。

- アプリ固有のドキュメントは各アプリ配下に置きます(例: デスクトップアプリのレビュー用チェックリストは [`../apps/desktop-app/docs/`](../apps/desktop-app/docs/))。
- リリース方針・アーキテクチャ全体像・コントリビューションガイドなど、リポジトリ横断の内容をここに追加してください。
